HTML & CSS

What is html? HTML (HyperText Markup Language) is the standard language used to create and design webpages. It structures web content using elements like headings, paragraphs, links, images, and lists, defined by tags enclosed in angle brackets.

HTML provides the backbone of a webpage. It uses elements like head, header, and body to define different parts of the page. The head holds metadata, styles, and scripts. The body contains the main content, such as paragraphs (p) and links (a). Together, these elements form the basic skeleton of a webpage.

To explain it to a kid: HTML is like the blueprint of a house, where each tag represents a different room or piece of furniture, giving structure and order to the space. Just as walls, doors, and windows define the layout of a home, HTML elements shape the way content like text, images, and links are arranged on a webpage, creating the framework for everything else to fit perfectly in place.

Container and Sectioning Elements

HTML has special elements to help organize content. Aside from the basic structure elements like head, header and body, you have other elements to help you section your HTML for general styling and self-guidance. You have the div, a versatile block container, while section, article, and nav provide semantic meaning. These tags improve both readability and accessibility, helping browsers and search engines understand your page layout.

To provide you with an example, I used a Div to specifically target the images of this portfolio's Homepage. This just made it easier for me to distinguish it from the rest of the elements in my messy HTML.

Style.CSS

While HTML is the content of a page, style.css is how that content looks like.

Although the content of a page may be sufficient in an informative way, it is the styling that allows for such information to be read comfortably. Style.css provides the means to frame every bit of content in your HTML however you want to, so that people visiting your page navigate through not only aided by lists and images, but also frames, space, and color.

Alignment & Justification

Just like in word document, you can edit the placement of items in your HTML, according to where you want them to be. Text and content alignment are key to good design. text-align, justify-content, and align-items are used to align elements horizontally and vertically. For example, justify-content: center; centers content within its container, creating a balanced layout across different screen sizes.

That is exactly what I did for my main content in the homepage. I used text-align: center; so the text would not start from the left side of the box, which would look less appealing in my opinion.

Pseudo-Classes & Links

CSS allows you to modify the appearance of links using pseudo-classes. a:hover, a:visited, and a:active let you change how links look based on their state (e.g., when clicked or hovered over).

For example, a:hover { color: red; } makes the link change color when hovered, enhancing interactivity. Every time you click a link that takes you to another page, the clicked link, now 'visited', will change its color to purple because it was edited with a:visited to reflect that it's been clicked.
